Redes:
- Para fazer a conexão de internet em diversos computadores usa-se o Backbone, que é um aparelho robusto e poucas existentes no mundo, que controla todas as conexões de internet em um só lugar. Dela saem diversas conexões que se espalham levando as informações.
- Cada país pode ter um ou mais provedor de acesso - que são as empresas telefônicas que contratam o serviço do backbone e repassam ao usuário final
	- Os serviços podem ser Dial-up, ADSL (banda larga), fibra ótica, rádio, satélite, móvel ou P2P (peer to peer)
- www -> DNS (transforma o URL em número) -> IP 

Protocolos:
- TCP/IP: Transmission Control Protocol e Internet Protocol
	4 camadas para transmissão de dados:
		1. Física (placa de rede)
		2. Rede (IP - estar na internet)
		3. Transporte (TCP ou UDP)
		4. Aplicação (FTP, SMTP - email, HTTP)
- TCP x UDP: os dois ja estão conectados quando vc tem internet
	UDP: rápida, não confiável (sem confirmação de recebimento), dispara as informações sem controle -> é usado para lives, pois pouco importa quem está recebendo e se está recebendo. 
	TCP: mais lento mas é voltado para conexão, confirma a comunicação de forma efetiva, mantém a integridade e ordem dos dados -> aplicativos de mensagens de texto, tem confirmação e ordem de dados
- Portas: são "portos" específicos onde as informações devem chegar de acordo com o seu objetivo:
	-20: FTP - porta para arquivos
	-22: SSH - porta segura para executar comandos
	-25 / 587 (internet moderna) / 465 (SMTP sobre SSL) / 2525 (Se a porta 587 estiver bloqueada, a porta 2525 é uma boa alternativa): SMTP - porta para mandar emails
	-53: DNS - porta para tradutor de URL para IP
	-80: HTTP - porta para requisição simples de internet
	-443: HTTPS / Apache- porta para requisição segura de internet
	-3306: MySQL
- Modem: Modulator-demodulator - aparelho que modula e demodula sinais. Hardware que converte dados em um formado que possa ser transmitido de um computador para o outro e lido por outro
- Roteador: Distribui a internet para um ou mais dispositivos. Pode fazer comunicação entre redes. Pode ser "burro" levando internet para todos de forma igual, mesmo sem necessidade.
- Switch: Distribui internet para um ou mais dispositivos de uma rede. Criado para ser inteligente, levando internet para quem "pede"
